# Semeando Trocas - Projeto Agrinho 2026

## 🎯 Objetivo
O **Semeando Trocas** é uma plataforma digital desenvolvida para fortalecer a economia circular no agronegócio local. O projeto visa conectar pequenos produtores rurais, permitindo a troca solidária de insumos excedentes (sementes, mudas, adubos), reduzindo o desperdício e diminuindo custos de produção.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica de páginas.
- **CSS3:** Estilização responsiva, Media Queries, Flexbox e Modo Escuro.
- **JavaScript (ES6):** Manipulação dinâmica do DOM, localStorage para persistência de dados, e lógica de negociação.
- **Font Awesome:** Biblioteca de ícones para interface profissional.

## 🚀 Como utilizar
1. Abra o arquivo `index.html` em qualquer navegador moderno.
2. Na tela inicial, insira seu nome para iniciar a sessão.
3. Utilize o formulário de "Cadastrar Nova Troca" para adicionar um novo item ao mural.
4. Navegue pelo mural e clique em "Propor Troca" para iniciar uma negociação.
5. Use o botão "Modo Escuro" no cabeçalho para alternar o tema da interface.
